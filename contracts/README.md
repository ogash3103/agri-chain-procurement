# AgriChain Procurement Smart Contract

This project is a basic agricultural procurement smart contract written in Solidity.

## Features
- Buyers can create orders
- Farmers confirm delivery
- Buyers release payment after delivery
- Escrow-style secure payment

## Contract Flow
1. Buyer creates order
2. Farmer confirms delivery
3. Buyer releases payment (ETH)

## Technology
- Solidity
- Remix Ethereum IDE
- Ethereum (Remix VM)
